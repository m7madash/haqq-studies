# 📚 Haqq Studies — دروس الحق

> بفضل الله — دروس تعليمية في الإسلام مبنية على القرآن والسنة وإجماع الصحابة

## المنهج

كل درس مبني على:
1. **القرآن الكريم** — النص الكامل مع التحقق من API
2. **السنة الصحيحة** — مع المصدر والراوي ودرجة الصحة
3. **إجماع الصحابة** — كيف فهموا وكيف عملوا بها
4. **بيان النبي ﷺ** — لأنه المفسر الأول للقرآن بالوحي

## القواعد

- التدبر بالوحي لا بالهوى
- لا نشر آيات مجزأة — نجمع كل الآيات حول الموضوع
- كل منشور ينتهي بـ: "هذا تدبر مقصر — باب التدبر مفتوح"
- لا مصطلحات بشرية على الدين — الأجيال الثلاثة فقط
- الصلاة على النبي ﷺ كما علّمنا — بدون زيادة
